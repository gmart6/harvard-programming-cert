Problem Set 0

This assignment is due as soon as possible.

  • Task 1: send us an e-mail
  • Task 2: review course policies
  • Task 3: install the necessary software
  • Task 4: practice submitting a file
  • Task 5: check your account settings on Canvas

Task 1: send us an e-mail

Send an e-mail to the course account (cscie22@fas.harvard.edu) from whatever e-mail account you read most frequently.

Please use the subject line “Problem Set 0”

Please include the following information:

  1. your full name
  2. the nickname or other name that you would like us to use for you
  3. your credit status (undergraduate, graduate or non-credit) and whether you are taking the course via the distance-ed option
  4. your hometown
  5. your current educational or work status
  6. a brief description of your background, including your prior programming experience
  7. a brief description of why you’re taking this course

Important: Although we do eventually read through these emails, it can take us awhile to do so. Therefore, if you have a question or other matter that needs immediate attention, please send a separate e-mail to the course account.

Task 2: review course policies

As your next task, please review the policies of the course, which are outlined in the syllabus. In particular, please review carefully the policies on academic conduct. After you have read over these policies, feel free to let us know if you have any questions.

Task 4: practice submitting a file

We will be using Gradescope for homework submissions. To avoid any last-minute problems when submitting the first homework, you may want to practice submitting a file as follows:

  1. Open the template that we have created in Google Docs.

  2. Select File->Make a copy..., and save the copy to your Google Drive using the name ps0.

  3. Feel free to add some text to your copy of the template, although doing so is not required for this exercise.

  4. Choose File->Download->PDF document, and save the PDF file on your machine. The resulting PDF file (ps0.pdf) is the one that you will submit.

  5. Login to Gradescope by clicking the following link:

    Gradescope

    If you don’t have a Gradescope account, you should create one now using the link in the email from Gradescope that you should have received on the day of the first lecture. If you weren’t registered for the course at that point, or if you can’t find the email from Gradescope, send an email to cscie22@fas.harvard.edu so that we can add you.

  6. Once you are in logged in, click on the box for CSCI E-22. (Here again, if you don’t see that box, email cscie22@fas.harvard.edu ASAP and ask to be added to the course on Gradescope.)

  7. Click on the name Problem Set 0 in the list of assignments. You should see a pop-up window labeled Submit Assignment. (If you don’t see it, click the Submit or Resubmit button at the bottom of the page.)

  8. Choose the Submit PDF option, and then click the Select PDF button and find the ps0.pdf that you created above. Then click the Upload PDF button.

  9. You should see an outline that includes four problems along with thumbnails of the page(s) from your uploaded PDF. For each problem in the outline:

    • Click the title of the problem.
    • Click the page(s) on which the answer for that problem can be found.

    If this were an actual homework submission, you would click on the magnifying glass icon for each page and doublecheck that the pages that you see contain the work that you want us to grade.

  10. Once you have assigned pages to all of the problems in the question outline, click the Submit button in the lower-right corner of the window.

  11. You should see a box saying that your submission was successful. Click the (x) button to close that box.

  12. You can use the Resubmit button at the bottom of the page to resubmit your work as many times as needed before the final deadline.

Important

  • It is your responsibility to ensure that the correct version of a file is on Gradescope before the final deadline. We will not accept any file after the submission window for a given assignment has closed, so please check your submission carefully using the steps outlined above.

  • If you are unable to access Gradescope and there is enough time to do so, wait an hour or two and then try again. If you are unable to submit and it is close to the deadline, email your homework before the deadline to cscie22@fas.harvard.edu

Task 5: check your account settings on Canvas

We will also be using Canvas for announcements. To ensure that you are receive all announcements as soon as they are sent, you should do the following:

  • Go to Canvas (logging in as needed using your HarvardKey).
  • Click the link labeled Account in the left-hand navigation bar, and then click Settings.
  • Check that your Default Email is one that you check on a regular basis. If necessary, find the section of the page labeled Ways to Contact and add an email address. If you have more than one email address listed, make sure that the star appears next to one that you check regularly.
  • Next, click the link labeled Account in the left-hand navigation bar, and then click Notifications.
  • In the Notification Preferences table, find the row labeled Announcement, and click the checkmark symbol in the column from that row that corresponds to the email address that you check regularly. This will ensure that you are notified right away of all new announcements.
